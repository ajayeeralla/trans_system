Mutually dependent inductive types: Bool and message Note that type Bool is different from the built-in type bool ilist: length-indexed list

Inductive ilist A : nat -> Type :=
| Nil : ilist A 0
| Cons: forall n, A-> ilist A n -> ilist A (S n).
